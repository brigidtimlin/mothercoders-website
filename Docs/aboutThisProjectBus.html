About the Project

This project uses a dataset from NYC OpenData that tracks school bus breakdowns and delays. The dataset contains real time updates, but the data I used for this graph was downloaded in March of 2019.

Learning Goals

For this project I want to refamiliarize myself with cleaning and manipulating a large dataset and use descriptive statistics to analyze the data. Additionally, I want to learn and practice some beginner D3.js skills. So far, I have imported the data into Stata, converted most of the string variables to integers, created dummy variables in place of inconsistently described categorical variables, and begun to clean up the various ways time is recorded in the dataset. I also built a simple bar graph using D3.js to visualize the number of times school buses break down or are delayed each school year.

Next Steps

I continue to work on cleaning this dataset, which has been challenging. For example, there are over 1700 unique ways time is inputted. As I continue this project I would like practice using Python for cleaning and analyzing. I want to visualize how different groups of students in different locations are impacted by delayed buses, and compute the total minutes per year that students on different routes spend waiting for, or stuck on, school buses.
